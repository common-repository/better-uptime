== Description ==

= The best uptime monitoring plugin for WP =

Better Uptime is a website uptime monitoring add-on. We call you when your website goes down.

Can’t afford downtime? Get alerted with the fastest uptime monitoring service.

= Better Uptime provides =

- Unlimited phone call alerts, as well as SMS, e-mail, Slack, Teams, and push notifications.
- 30-second multi-location checks. Each incident is verified from at least 3 locations before we proceed to alert you.
- Detailed incident timeline describing exactly what happened. This includes screenshots and error logs for every incident.
- Built-in incident management with on-call duties and automatic escalation.
- Scheduling of on-call rotations directly from Google Calendar.
- Beautiful status pages on your own sub-domain in just a few clicks. Customize design and set it up in 5 minutes.
- Over 100+ integrations with tools you already use.
